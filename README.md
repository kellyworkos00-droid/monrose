# Monross Hardware

Monross Hardware is a responsive React and Vite storefront concept for a product-selling brand. The UI uses yellow as the dominant visual identity with red accents for offers, highlights, and calls to action.

## What is included

- A bold landing page with product-focused hero content
- Featured product cards and selling-focused sections
- Responsive layouts for desktop, tablet, and mobile
- A floating bottom navigation bar for mobile screens
- A VS Code task named `Run Monross Hardware` for local development

## Scripts

- `npm run dev` starts the local Vite development server
- `npm run build` creates the production build in `dist`

## Notes

- The contact email in the footer is placeholder content and should be replaced with the client's real address.
- Product visuals are represented with gradient-based cards so the project works without external image assets.
